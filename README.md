# Interactive Sales Dashboard in Excel

## Overview
This project demonstrates an interactive sales dashboard built using Microsoft Excel.

## Features
- Interactive Slicers
- Timeline Filter
- KPI Cards
- Monthly Sales Trend
- Sales by Product
- Sales by Location
- Top 5 & Bottom 3 Selling Products

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
